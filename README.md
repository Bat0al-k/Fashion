# Fashion Platform — E-Commerce Website  

A modern **e-commerce web application** built with **Angular 17** for the frontend.  
This project allows users to browse, shop, and manage fashion products seamlessly with authentication, user profiles, and order management.

---

## 🖥️ Tech Stack

### 🔹 Frontend
- **Angular 17 (Standalone Components + Lazy Loading)**
- **TypeScript**
- **Angular Router**
- **Reactive Forms**
- **ngx-toastr** for alerts and notifications  
- **Bootstrap** for responsive UI  
- **Interceptors** for token & error handling  

## 🚀 Features

### 👤 Authentication
- Register / Login with JWT or google
- Email verification  
- Reset password  

### 🛍️ Products
- View all products  
- Filter & sort by category, price, or popularity  
- Product details page  

### 🧺 Cart & Checkout
- Add / remove products from cart  
- Update quantity  
- Checkout process with order summary  
- Integration with Stripe
